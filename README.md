# msipenko dotfiles

## Setup

1. Updates macOS to latest version.
2. Install xcode.

```sh
xcode-select --install
```

3. Clone this repo

```sh
git clone https://github.com/martinssipenko/dotfiles.git ~/src/msipenko/.dotfiles
cd ~/src/msipenko/.dotfiles
```

4. Run bootstrap script
```sh
source bootstrap.sh
```
